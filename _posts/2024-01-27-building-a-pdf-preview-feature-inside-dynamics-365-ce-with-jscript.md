<img border="0" data-original-height="400" data-original-width="700" height="183" src="{{ site.baseurl }}/assets/images/034/img_647b95229f.png" style="display: none;" width="320" /><p style="text-align: justify;"><span>&nbsp;&nbsp; &nbsp;</span>I recently encountered a question regarding the possibility of previewing a PDF file inside Dynamics 365 CRM or a Model Driven App without opening an additional tab or window. After conducting some research, I found that there is no out-of-the-box (OOTB) feature for previewing PDF files. However, I devised an approach to achieve this by leveraging client-side scripting, specifically using JScript.</p><p></p><p style="text-align: justify;">One of the methods I explored is the <span style="background-color: #eeeeee; font-family: Inconsolata;">Xrm.Navigation.openFile(file, openFileOptions)</span> script. This script allows you to open a file in another tab or window. However, the goal is to have the file open inline within the Unified Client Interface (UCI) page. More details on this script can be found in the <a href="https://learn.microsoft.com/en-us/power-apps/developer/model-driven-apps/clientapi/reference/xrm-navigation/openfile" rel="nofollow" target="_blank">Microsoft Docs</a>.</p><p style="text-align: justify;">To implement this, I created a file field on the Accounts Entity named "Test File" and added it to the main form. The preview feature is intended for files uploaded to this column.</p><div class="separator" style="clear: both; text-align: center;"><div class="separator" style="clear: both; text-align: center;"><a href="https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEgpMvPAFnnqC0tjQrcfyvhFpE_67Vhej5G8Uqdb2xM4L0hsa25xwZAx025eO4HRnPikWQyHp07mKwmWxRo1lvuDmUp96Og80ku6WyWd8v1gY5hgUyntjKmsTWQD7rfWkj1sTuQ0QaPpqoVruG56Flq_ekYIXqMXXtGDUYOa1pbt9Q_wODP0aQY6mpW8USQ/s611/field-on-main-form.png" style="margin-left: 1em; margin-right: 1em;"><img border="0" data-original-height="611" data-original-width="518" height="400" src="{{ site.baseurl }}/assets/images/034/img_224d66c694.png" width="339" /></a></div></div><p style="text-align: justify;">To accomplish the preview functionality, I decided to create an HTML web resource and a JS web resource.
