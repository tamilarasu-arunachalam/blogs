However, it's important to note that the Dataverse API retrieves a maximum of 5000 records per request. If the number of records exceeds this limit(of 5000), the API will provide a 'next link' field in the response. Utilizing this 'next link' value, subsequent requests can be made to fetch the next set of 5000 records. This process continues until all records are fetched.
