<img alt="Import records with alternate lookup references in Dynamics 365 CE" data-original-height="400" data-original-width="700" height="183" src="{{ site.baseurl }}/assets/images/019/img_868419228a.png" style="display: none;" width="320" /><p style="text-align: justify;"><span>&nbsp;<span>&nbsp;&nbsp; &nbsp;</span></span>If you are in a place of importing bulk data into the Dynamics 365 CE, you have many choices like importing as excel, CSV or XML. But if you are having records for two tables which are related to each other as like accounts and contacts. You have to be careful while importing those records because this import may cause the following error “<b><i>The lookup reference could not be resolved</i></b>”. The error is due to the lookup values which are not present in the table we are looking for. This blog will guide to import the related records at a single stretch.</p><p style="text-align: center;"></p><div class="separator" style="clear: both; text-align: center;"><a href="https://blogger.googleusercontent.com/img/a/AVvXsEh6aHxSIJ3zkYsokIwz0wHoCTjVbGkb7XcMUknkCyUSEgQIjTCthizzIz6ljUcqEd9-4d_6DHyh95uRy8DncCT3xByclud8AUi8LdHfP0BV7e1HqeSyV66nrzJxzEStSJkzTUzF782TpwCdeXu6hAdsvczGN7e_smGOVSHaIM6xnBoXmzpLgS6bFQnZ" style="margin-left: 1em; margin-right: 1em;"><img alt="" data-original-height="576" data-original-width="626" src="{{ site.baseurl }}/assets/images/019/img_8a34e78eb5.png" /></a></div><p></p><p style="text-align: justify;"><span>&nbsp;&nbsp; &nbsp;</span>For importing records with alternate lookup references, we have to get the data for both the tables. In our case, we have taken accounts and contacts table. Go to the <a href="https://admin.powerplatform.microsoft.com/environments/" rel="nofollow" target="_blank">Environment</a>&nbsp;➔ Settings ➔ Data Management ➔ Data Import Wizard. The wizard will open in the new tab. Click on the Import Data button in the wizard, a dialog box opens up for importing files.</p><p></p><div class="separator" style="clear: both; text-align: center;"><a href="https://blogger.googleusercontent.com/img/a/AVvXsEhKXgzk5x6or_X5tKeHg2UFMzqyM5bVlrvR8dHmA9UNOQDvfZU53RqX6CbUqX_QujuhaIGYfyjRe6tbstv_FcQMFkLRET067IXl7CI-Z-whTPai8paSYwMRD2bgjaUSxUIOWas-9pSlwZOp3nPfHVVk9XFbKm1ad1T0-PufXHT2SNxcxqdqRutf4X82" style="margin-left: 1em; margin-right: 1em;"><img alt="" data-original-height="240" data-original-width="601" src="{{ site.baseurl }}/assets/images/019/img_0329e8609f.png" /></a></div><br /><div style="text-align: justify;"><span>&nbsp;&nbsp; &nbsp;</span>For importing, I have two CSV files, one contains data for accounts and another one has for contacts. I have zipped these two files, so that I can upload it as a single file. Click on choose file and select the zip file which we zipped before and click on next.</div><p></p><p></p><div class="separator" style="clear: both; text-align: center;"><p style="clear: both; text-align: center;"><a href="https://blogger.googleusercontent.com/img/a/AVvXsEjqhNMhYk8KWN5Hr95OBVdKh_Gu3AzlMAkmTe8blHPAQqe1Jm44pF8YSQlIQVtzui2QL7M4aJE4BffFVArSJz97MkgBrNWvL1TRipnJIB4xeAphoCh69K1M9V6NKV9PvioZSLKa6keiPwafjDOSbCwLJZqJH6ppONQ91sQ4p6eWcDBByfPsBYnzbmzb" style="margin-left: 1em; margin-right: 1em;"><img alt="" data-original-height="862" data-original-width="898" src="{{ site.baseurl }}/assets/images/019/img_51498994a2.png" /></a></p><div style="text-align: left;"><p style="clear: both; text-align: justify;">You will get to a Review File Upload Summary, in which you can make sure your CSV files are uploaded and set the delimiter. As I uploaded a CSV file, so the delimiter was comma(,). Then click on Next.</p><p style="clear: both; text-align: center;"><a href="https://blogger.googleusercontent.com/img/a/AVvXsEiZpZyT4YpfMe87osn9Mi9zuqTa7sR9smz8MPf3fG4Un8XsiL3iiTewqiiYmrtzS4_TPnCfLIzIx4-ULzCflkcToPTxMjsSZyFETg0QHmwCH6lVEVZYv4k-xkmZECGarEqqkNJdyqKZsaH5nSnvg0qrH2TX1bm7n0_tLMQhS3btvz733csRvAIDXlfr" style="margin-left: 1em; margin-right: 1em;"><img alt="" data-original-height="616" data-original-width="640" src="{{ site.baseurl }}/assets/images/019/img_d9e3d946c3.png" /></a></p><p style="clear: both; text-align: justify;">Map the source files with the dataverse table which you want to import. Ignore if it is correctly mapped already.&nbsp;<span style="text-align: left;">It will take you to the Data map page where you can see two types of mappings, Default and SampleDataMap. Select Default Mapping and click Next.</span></p><div class="separator" style="clear: both; text-align: center;"><a href="https://blogger.googleusercontent.com/img/a/AVvXsEgz-X9DBnZRlyIaGZLUe_WmgWSZn59BoQAYhrWsXsbvZYyLTlN3d7qBRpY-B9m7CHubiP8ED5TWyRz44PeWA9g8SEiUTCHKrfiGHUNwYQ7OShoxHe670C1V2aR0I7QrifIhW97yASNgAQL2AfLAP9arHU94ohzlQe1GiZRQKrSd5Q3wF8ZhzX8f0rj2" style="margin-left: 1em; margin-right: 1em;"><img alt="" data-original-height="325" data-original-width="858" src="{{ site.baseurl }}/assets/images/019/img_105983cc22.png" /></a></div><p style="clear: both; text-align: justify;"></p><div class="separator" style="clear: both; text-align: justify;">In the mapping page, make sure all the fields are mapped correctly. For accounts table, the primary contact (lookup) field should be mapped with the full name field of the contact table.</div><div class="separator" style="clear: both; text-align: center;"><div class="separator" style="clear: both; text-align: center;"><p style="clear: both; text-align: center;"><a href="https://blogger.googleusercontent.com/img/a/AVvXsEhYV2eYlJuxH2CwZ4TTloV9eU8s01_beuPXi1Vq6eyDeJrnWvtSh3FL-ygF0Uw7pMcrKQQqN0r18b0uO_ZrfA8iWqkbtcKbV6SNg3e044VB7ZG1skUQ5KZPZ-LA-scsqqFMn_251Bs6kgvUC6CI0CJ3JMOxQ_BQ0w5Zy0XlX4S6kkTMXLk1aAbU8GcR" style="margin-left: 1em; margin-right: 1em;"><img alt="" data-original-height="560" data-original-width="787" src="{{ site.baseurl }}/assets/images/019/img_30417ec007.png" /></a></p><p style="clear: both; text-align: justify;"></p><div class="separator" style="clear: both; text-align: left;">For the contact table, the Company name (lookup) field should be mapped with the account name field in accounts table.</div></div><p style="clear: both; text-align: center;"><a href="https://blogger.googleusercontent.com/img/a/AVvXsEhQCXlQolmIwDOH9QUaIJmjsTuMliqmQYTTAj-qd7fTgKywwz5Lpxf6jtWqpnDS5B-jIWPyFEfC3oDg6WiomZ4b0utHViRgsZYy8Cudsayafc7YemzKKfbQ_OhwjgAbasrfM4vIn7qyRCAur5xrGchAJCSBElqXJEhn3b0ZjkaOM0Q8lo1-AeUAZM9r" style="margin-left: 1em; margin-right: 1em;"><img alt="" data-original-height="566" data-original-width="787" src="{{ site.baseurl }}/assets/images/019/img_5cc47399fe.png" /></a></p><div class="separator" style="clear: both; text-align: center;"><p style="clear: both; text-align: justify;">Click on Submit to finish the import.
